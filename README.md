# Maison Romantique

Hjemmeside for chambre d'hôtes i Olargues, Frankrig.

## Brug med GitHub Pages
1. Opret et nyt repository (f.eks. `maison-romantique`).
2. Upload `index.html`, mappen `images/` og denne `README.md`.
3. Gå til **Settings → Pages**, vælg branch `main` og mappe `/ (root)`, og gem.
4. Siden er live på `https://<brugernavn>.github.io/maison-romantique/` efter et øjeblik.

## Redigér indhold
Alt indhold ligger i `index.html`. Det vigtigste at vide:
- **Mail og telefon:** søg efter `mailto:` og `tel:+4528100751` og indsæt den rigtige mailadresse.
- **Billeder:** udskift filer i `images/` (behold samme filnavne), eller ret stierne i HTML.
- **Booking-dato:** teksten "fra den 7. oktober" står i sektionen med `id="booking"` og i footeren.

## Mappestruktur
- `index.html` — hele siden (ét fil-design, ingen afhængigheder)
- `images/` — weboptimerede billeder
